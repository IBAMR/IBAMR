Major Changes

Minor Changes

  1. Fixed: Fixed upper convected derivative in cases of non divergence-free velocity fields.
    (Aaron Barrett, 2025/04/24)
  2. Fix: Updated the IIM velocity-interpolation scheme to the version proposed by Tan et al. in “An Immersed Interface Method for the Incompressible Navier–Stokes Equations with Discontinuous Viscosity Across the Interface.” (Qi Sun, 2025/06/09)
  3. Improved: The CMake configuration now exports basic information about our external dependencies.
    (David Wells, 2025/07/21)
  4. Fixed: The H5Part marker point output now saves the marker ids.
    (David Wells, 2025/08/11)
  5. Fixed: Fixed the ability of IBExplicitHierarchyIntegrator to restart any marker points it owns.
    (Marshall Davey, 2025/08/14)
  6. Fixed: Fluid sources and sinks are now computed before regridding. This fixes a bug where sources were not set, which caused arbitrary values to be added to the divergence, which in turn caused arbitrary values in the velocity field. Furthermore, IBFEMethod now assumes that sources may be computed at any point in a timestep (or outside of a timestep).
    (David Wells, 2025/08/28)
  7. New: More composite B-spline kernels.
    (Aaron Barrett, 2025/05/218)
  8. New: Added interpolation functions to easily interpolate Eulerian data to specified positions.
    (Aaron Barrett, 2025/05/29)
