Incompatibilities

  1. Changed: The complex fluid solver has been revamped to allow for more flexible specifications of fluid models. The class CFRelaxationOperator has been replaced with CFStrategy, in which users can specify both a relaxation function and a conformation-to-stress conversion function. Additional solver options have also been added to CFUpperConvectiveOperator.
    (Aaron Barrett, 2024/02/01)

Major Changes

  1. New: Implementations of AllenCahn and enthalpy methods for the simulation of phase change problems with volume change. (Ramakrishnan Thirumalaisamy, 2024/01/30)
  2. Improved: Many quantities which did not need to be in the restart database have been removed from it. This decreases the size of the restart files by over 50%.
    (David Wells, 2024/03/11)
  3. Enhancement: Remove the volume shift algorithm and replace it with a volume redistribution algorithm in the RelaxationLSMethod class.
    (Amneet Bhalla, Kaustubh Khedkar, and Ramakrishnan Thirumalaisamy, 2024/04/04)
  4. New: Introduce a new LevelSetUtilities class to provide implementations of helper functions for enforcing mass/volume conservation across phases. This class also includes implementations of SetLSProperties (to set or reset the level set after reinitialization) and TagLSRefinementCells (to tag cells for grid refinement based on the level set). These additions reduce duplicated codes in multiphase examples.
    (Amneet Bhalla, Kaustubh Khedkar, and Ramakrishnan Thirumalaisamy, 2024/04/04)
  5. Enhancement: Splitting BrinkmanPenalization into normal and tangential components near the solid interface.
    (Amneet Bhalla, Kaustubh Khedkar, and Ramakrishnan Thirumalaisamy, 2024/04/04)

Minor Changes

  1. New: Implementations of composite 6/5 and 5/4 B-spline kernels. (Boyce Griffith, 2024/01/29)
  2. Bug fix: Fixed masking function for StaggeredPhysicalBoundaryHelper. (Aaron Barrett, 2024/03/01)
  3. Improved: Added a 'make indent' target under CMake for automatically formatting all source code. This check is also run by the CI. (David Wells, 2024/03/07)
  4. Fixed: Fixed a compatibility problem where IBAMR didn't compile correctly with older versions HDF5 which use MPI. (David Wells, 2024/03/08)
  5. Enhancement: The new header vc_ins_utilities.h sets fluid properties such as density and viscosity for both two-phase and three-phase flows throughout the entire domain. Additionally, this class provides implementations for gravity force calculations in two-phase and three-phase flows. (Ramakrishnan Thirumalaisamy, 2024/04/22)
  6. Improved: Classes derived from INSHierarchyIntegrator only compute the vorticity when they need to instead of at the end of every time step.
    (David Wells, 2024/05/15)
