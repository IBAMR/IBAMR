Incompatibilities

  1. Changed: Fixed implementations of CellNoCornersFillPattern and SideNoCornersFillPattern, and updated the class constructors to correspond to the new implementation.
    (Boyce Griffith, 2025/09/23)
  2. Changed: Added input databases to the constructors for several operators and preconditioners. These enable runtime configuration of coarse-fine interface discretization details.
    (Boyce Griffith, 2025/10/02)
  3. Changed: IBAMR now requires IBSAMRAI2 version 2025.10.29.
    (David Wells, 2025/10/29)

Major Changes

  1. Improved: The doxygen manual is now integrated into the CMake build system and also creates documentation for all SAMRAI classes. (David Wells, 2025/11/12)
  2. Improved: Several communication patterns no longer communicate duplicate data: see Incompatibilities for more information.
    (Boyce Griffith, 2025/09/29)
  3. Improved: LEInteractor now allocates significantly fewer temporary arrays.
    (David Wells, 2025/10/27)
