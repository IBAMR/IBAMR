Incompatibilities

  1. Changed: IBAMR now requires C++17.
    (Boyce Griffith, 2024/07/20)
  2. Changed: IBAMR now requires Eigen 3.4 or newer.
    (David Wells, 2025/01/09)
  3. Changed: The minimum supported version of libMesh is now 1.7.0.
    (David Wells, 2025/01/29)

Major Changes

  1. New: added IMEX-BDF2-type timestepping (similar to VSSBDF2 in this preprint).
    (Boyce Griffith and Aaron Barrett, 2024/08/19)

Minor Changes

  1. New: added IBTK::IndexUtilities::getCellCenter().
    (David Wells, 2024/03/19)
  2. New: added an option to skip the velocity solve in the projection preconditioner.
    (Boyce Griffith, 2024/09/07)
  3. Fixed: Fixed issue with time-dependent boundary value updates for plotting. Before, only the intial boundary values were used. Now they update appropriately in time. (Cole Gruninger, 2024/06/26)
