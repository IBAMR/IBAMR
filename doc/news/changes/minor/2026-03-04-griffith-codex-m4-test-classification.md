Refined Milestone 4 parity-test organization by separating self-contained CI coverage from external-reference verification workflows. The standalone CAV parity tests in `tests/navier_stokes` were renamed to `*_semantics_parity` and expanded to run in both 2D and 3D, while MATLAB/Octave-dependent verification inputs and tooling were moved under `tests/external-reference/cav-stokes-ib/` with dedicated manual CMake targets. This keeps routine attest coverage deterministic and self-contained while preserving a simple developer path for external-reference parity runs without committing `.ref` data files.
