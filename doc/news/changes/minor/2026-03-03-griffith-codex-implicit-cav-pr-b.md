Added solver wiring for coupling-aware ASM subdomain construction in `StaggeredStokesPETScLevelSolver` via an input-driven `asm_subdomain_construction_mode` selector with enum string conversion support. The solver now chooses exactly one ASM subdomain construction path per setup (geometrical or coupling-aware), and in coupling-aware mode builds subdomains from the A00 velocity block extracted from the solver-owned level matrix. This update also adds deterministic navier_stokes attest coverage for solver-mode ASM subdomain generation.
